
# CRM

## COMO MONTAR EL PROYECTO

Crear un archivo `.env` con las siguientes variables, 

```bash
PORT=3000

```

Para prender el proyecto solo realizar el comando `npm run dev`

## LIBRERIAS

```bash

axios=1.7.7
dotenv=16.4.5
express=4.21.0
morgan=1.10.0
nodemon=3.1.7

```
